#List-stack Object

The `list-stack` object removes the bullets and indent from lists.

	<ul class="o-list-stack">
		<li></li>
		<li></li>
		<li></li>
	</ul>

## Installation

Install using Bower:

	$ bower --save bare-list-stack

Import into project:

	@import "bower_components/bare-list-stack/objects.list-stack";
